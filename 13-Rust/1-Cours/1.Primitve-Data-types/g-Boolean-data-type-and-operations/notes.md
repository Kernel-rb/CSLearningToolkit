# Boolean Data types
    - Value can be either True or False
    - Logical operations can be performed on boolean values.
    - 1 -> true 
    - 0 -> false

## And 
    - False & True -> False
    - False & False -> False
    - True & True -> True
    - True & False -> False

## Or 
    - True | False -> True
    - True | True -> True
    - False | True -> True
    - False | False -> False

## Short-Circuiting
    - false && [anything] -> false
    - true || [anything] -> true
